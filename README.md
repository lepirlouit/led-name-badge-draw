# Pixel Draw

## usage
change line in file for initial content:
```
drawing = bytearray([0x00, 0x18, 0x18, 0x00, 0x38, 0x18, 0x18, 0x18, 0x18, 0x3c, 0x00])
```
run with
```
python draw-11x8.py
```
